SETS

Easy Reviewer for Grade 5 Learners

1. What is a Set?

A set is a group or collection of objects. The objects inside a set are called elements.

Example: A set of colors of the Philippine flag is {red, white, blue}.

Sets are usually named using capital letters like A, B, C, U.

We write the elements inside curly braces: { }

2. Important Set Words

Element

An object inside a set.

Example: In A = {cat, dog}, cat is an element of A.

Cardinality

The number of elements in a set.

If A = {red, white, blue}, then n(A) = 3.

Empty Set / Null Set

A set with no elements.

Written as { } or ∅.

Equal Sets

Sets with exactly the same elements.

{A, B, C} = {C, B, A}

Equivalent Sets

Sets with the same number of elements.

{red, blue} and {cat, dog} are equivalent because both have 2 elements.

Universal Set

The set of all objects being talked about in the problem.

Usually written as U.

3. Three Ways to Describe a Set

1. Roster or Listing Method

List every element inside curly braces.

A = {1, 2, 3, 4, 5}
2. Verbal Description Method

Describe the set using words.

A is the set of counting numbers from 1 to 5.
3. Set Builder Notation

Describe the set using a rule.

A = {x | x is a counting number from 1 to 5}

4. Well-Defined Set

A set is well-defined if we can clearly decide whether an object belongs to it or not.

Well-defined:
A = {1, 3, 5, 7, 9, ...}
This is clear because it means odd counting numbers.
Not well-defined:
Set of intelligent students
This is not clear because people may have different opinions.

5. Subset and Proper Subset

A is a subset of B if every element of A is also in B.

A = {1, 2}
B = {1, 2, 3}
Since 1 and 2 are also in B, A ⊆ B.

Proper subset means A is inside B, but B has at least one extra element.

A = {1, 2}
B = {1, 2, 3}
A is a proper subset of B: A ⊂ B.
Remember:
  • Every set is a subset of itself.
  • The empty set { } is a subset of every set.
  • For a set with 3 elements, there are 8 subsets.
  • For a set with 4 elements, there are 16 subsets.

6. Complement of a Set

The complement of set A is written as A'. It means all elements in the universal set U that are not in A.

U = {1, 2, 3, 4, 5, 6, 7}
A = {2, 4, 6}
A' = {1, 3, 5, 7}
